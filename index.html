    public final Timer crossDCLatency;

        crossDCLatency = Metrics.timer(factory.createMetricName("CrossNodeLatency"));
    }
    @Override
    public int getCacheSize() {
        // TODO Auto-generated method stub
        return 0;
    }
    @Override
    public void setCacheSize(int size) {
        // TODO Auto-generated method stub

    }

}

3.register MBean

public class Agent {
    public static void main(String[] args)
            throws MalformedObjectNameException, NullPointerException, InstanceAlreadyExistsException,
            MBeanRegistrationException, NotCompliantMBeanException, InterruptedException {
        MBeanServer server = ManagementFactory.getPlatformMBeanServer();
        ObjectName name = new ObjectName("com.fs.metrics:type=CrossDCLatency"); 
        Latency mbean = new Latency();
        server.registerMBean(mbean, name);

        System.out.println("waiting here for remote management...");
        Thread.sleep(Long.MAX_VALUE);
    }
}

6.mean 等值的计算

final AtomicLongArray buckets
private final long[] bucketOffsets

AtomicLongArray 长整型数组原子操作
将值坐标为[[1, 2, 3, 4, 5, 6, 7, 8, 10, 12, 14, 17, 20, 24, 29, 35, 42, 50, 60, 72, 86, 103, 124, 149, 179]
这样
先判断n在这里面，如果在相应位置+1
如果n不在，使用Array.binarySearch()找到n离 序列中最近的一个最大的数，然后+1
所以这边的mean计算值比通常意义上的mean值要大

redhat7 上安装dummynet

更多请访问 http://www.webpersonaldeveloper.cn

摘要：

在redhat 上部署dummynet 需要将ipfw 编译为内核模块，而ipfw需要调用linux kernel的一些函数。
linux kernel版本在不断提升，相关的数据结构也在变化。所以可能需要根据你要部署的系统内核版本，来调整参数

提醒

如果你要部署在Redhat6,CentOS6等内核版本比较低的平台，不必参考下面的部署流程。
这篇文章就可以满足的你要求
http://xmodulo.com/how-to-install-dummynet-on-centos.html

如果遇到问题，再返回这篇进一步学习
