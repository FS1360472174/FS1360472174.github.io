git 常用命令

1.apple patch

git apply xxx.patch

2.clone 指定的branch

git clone -b

3.查找本地branch

git branch

4.更换repo远程地址
git remote get-url origin //查看远程url
git remote set-url origin //设置新的url
git push // 提交到新的repo
